# Welcome To My CrowdStreet Registration Repo

#steps to run on your local machine
- open terminal
- cd into folder you want your repo
- clone repo from github
- in terminal, git clone and paste file you cloned from Github
- open text editor 
- open new window and navigate to where you put your cloned repo 
- press open and the cloned repo should be available in your text editor
- in terminal you have three options when it comes to running Cypress
